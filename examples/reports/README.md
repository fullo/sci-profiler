# Reports

Sample SCI profiling reports generated by [`generate_reports.ts`](../generate_reports.ts).

## Regenerate

```bash
npx tsx examples/generate_reports.ts
```

## Files

### Benchmark Reports

| File | Format | Description |
|------|--------|-------------|
| [benchmark.md](benchmark.md) | Markdown | Sample operations (sort, parse, hash, regex) as a table |
| [sci-profiler.jsonl](sci-profiler.jsonl) | JSONL | Same data in PHP-compatible flat format, one JSON object per line |

### Profiler Overhead Reports

| File | Format | Description |
|------|--------|-------------|
| [profiler-overhead.md](profiler-overhead.md) | Markdown | Per-call cost of profiler functions in μs and μgCO₂eq |
| [profiler-overhead.jsonl](profiler-overhead.jsonl) | JSONL | Self-profiling data in PHP-compatible flat format |

## Analyzing JSONL

The `.jsonl` files use the same flat dot-notation format as [sci-profiler-php](https://github.com/fullo/sci-profiler-php):

```bash
# Show tool name and SCI score
cat examples/reports/sci-profiler.jsonl | jq -r '[.tool, .["sci.sci_mgco2eq"]] | @tsv'

# Sum total carbon
cat examples/reports/sci-profiler.jsonl | jq -s '[.[]["sci.sci_mgco2eq"]] | add'

# Find the most expensive operation
cat examples/reports/sci-profiler.jsonl | jq -s 'sort_by(.["sci.sci_mgco2eq"]) | reverse | .[0] | {tool, sci: .["sci.sci_mgco2eq"]}'

# Merge with PHP profiler output for full-stack analysis
cat examples/reports/sci-profiler.jsonl /tmp/sci-profiler/sci-profiler.jsonl | jq -s '[.[]["sci.sci_mgco2eq"]] | add'
```
